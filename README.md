## Project Structure
── pdfmanager

│   ├── core

│   ├── cli

│   ├── gui

## How to build
You can build both the cli and the gui using:
```
./gradlew build
```

Or build a project individually using:
```
./gradlew pdfmanager:{project}:build
```

## How to run 
```
./gradlew pdfmanager:{project}:run
```
